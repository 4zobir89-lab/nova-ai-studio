## 🚀 Quick Start

### Prerequisites

- Node.js ≥ 20
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/4zobir89-lab/nova-ai-studio.git
cd nova-ai-studio

# Install dependencies
npm install

# Start development
npm run dev
```

### Usage Example

```typescript
import { NovaStudio } from '@nova-ai/studio';

const nova = new NovaStudio();

// Provide a project idea
const project = await nova.createProject({
  idea: "Build an e-commerce platform for luxury fashion",
  requirements: {
    target: "High-end fashion retailers",
    budget: "enterprise",
    timeline: "3 months"
  }
});

// Nova automatically:
// 1. Analyzes the idea
// 2. Creates a plan
// 3. Assigns 15 agents
// 4. Builds the complete product
// 5. Deploys to production

console.log(project.status); // "completed"
console.log(project.url);    // "https://my-luxury-store.com"
```

## 🛡️ Security

Security is built into every layer:

- **Secrets Management** — Never hardcode credentials
- **Agent Permissions** — Each agent has strict boundaries
- **Input Validation** — All user input is sanitized
- **Audit Logging** — Every action is tracked
- **Sandbox Mode** — Agents run in isolated environments

See [SECURITY.md](SECURITY.md) for details.
